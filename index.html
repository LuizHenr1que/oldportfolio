HTML

HTML abreviação para a expressão inglesa HyperText Markup Language, que significa: "Linguagem de Marcação de Hipertexto" é uma linguagem de marcação utilizada na construção de páginas na Web. Em outras palavras, o HTML é o esqueleto do nosso site.

JavaScript

JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma. O JavaScript tem a responbilidade de dar interação para nossa página.

CSS

Cascading Style Sheets é um mecanismo para adicionar estilo a um documento web, ou seja, é o CSS que torna a nossa página algo bonito, com ele podemos atribuir cores, tamanhos, alinhamentos, formatações de texto e muito mais!

BOOTSTRAP

Bootstrap é um framework web com código-fonte aberto para desenvolvimento de componentes de interface e front-end para sites e aplicações web usando HTML, CSS e JavaScript, baseado em modelos de design para a tipografia, melhorando a experiência do usuário em um site amigável e responsivo.
